# Prova React

Aplicação desenvolvida como desafio técnico em React, com foco em componentização, gerenciamento de estado, persistência de dados e testes end-to-end.

## Tecnologias

- React
- TypeScript
- Redux Toolkit
- Redux Persist
- Axios
- Styled Components
- Cypress

## Sobre o projeto

O projeto simula uma aplicação com fluxo de navegação, seleção de itens, gerenciamento de estado global e persistência dos dados. Foi desenvolvido como exercício prático para consolidar conceitos de React e organização de aplicações front-end.

## Como rodar

npm install
npm start

## Testes

npm run cypress:open
